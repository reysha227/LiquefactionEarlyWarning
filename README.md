# Liquefaction Early Warning System

Proyek ini merupakan aplikasi .NET MAUI untuk sistem peringatan dini potensi likuefaksi tanah berbasis monitoring tekanan air pori, vibrasi tanah, Fuzzy Logic, Neural Network, NLP, SQLite, dan face scan.

## Fitur Utama
- Registrasi dan login pengguna
- Verifikasi wajah sebelum masuk dashboard
- Dashboard monitoring risiko likuefaksi
- Simulasi data tekanan air pori, vibrasi tanah, dan ground acceleration
- Analisis risiko menggunakan Fuzzy Logic
- Prediksi risk score menggunakan Neural Network
- Penjelasan mitigasi menggunakan NLP API
- Penyimpanan data lokal menggunakan SQLite
- Dokumentasi UML, flowchart, jurnal, HKI, dan hasil simulasi

## Struktur Portofolio
- `source-code/` berisi source code aplikasi .NET MAUI
- `docs/` berisi dokumen paper, novelty, UML, flowchart, jurnal, dan HKI
- `screenshots/` berisi tampilan aplikasi
- `simulation/` berisi hasil simulasi JPG dan XLSX
- `presentation/` berisi PPT presentasi final

## Tema
Kebencanaan, khususnya sistem peringatan dini potensi likuefaksi tanah.

## Teknologi
.NET MAUI, C#, SQLite, Fuzzy Logic, Neural Network, NLP API, Face Scan.
